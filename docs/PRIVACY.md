# 應用程式隱私權政策

## 生效日期：YYYY/MM/DD

本應用程式（以下稱「本 App」）重視使用者隱私，承諾依據相關法律保護您的個人資料。本隱私權政策說明我們如何收集、使用、儲存及保護您的資訊，以及您的權利。

1. 收集資訊的類型

- 個人識別資訊：如姓名、電子郵件、電話號碼（僅在使用特定功能時收集）。

- 使用資訊：使用本 App 的操作紀錄、訪問時間、功能使用情形。

- 裝置資訊：裝置型號、作業系統版本、IP 位址、裝置識別碼等。

- 第三方服務資訊：如透過 Google、Apple 或 Facebook 登入所提供的基本資料。

2. 資訊的使用目的

- 提供、維護及改進本 App 功能與服務。

- 回應使用者問題或支援需求。

- 傳送服務通知、更新或行銷訊息（僅在取得同意後）。

- 分析使用模式以優化使用體驗與安全防護。

3. 資訊的分享與揭露

- 不會將您的個人資訊出售給第三方。

- 在以下情況可能分享資訊：
  - 法律要求或司法程序所需。

  - 維護本 App 的安全與防止詐騙行為。

  - 與合作夥伴共享，用於技術支援或分析，但會經過匿名化處理。

4. 資訊的儲存與保護

- 使用業界標準的技術與管理措施保護個人資料。

- 儲存期限僅限於達成收集目的所需的期間，或依法律規定。

- 個人資料將在用途終止或使用者請求刪除時進行刪除或匿名化。

5. 使用者權利

- 查詢或請求存取您的個人資料。

- 請求更正不正確或不完整的資料。

- 請求刪除您的個人資料（法律另有規定者除外）。

- 撤回同意或拒絕特定資料處理。

- 選擇不接收行銷或推播通知。

6. 兒童隱私
   本 App 不會有意收集 13 歲以下兒童的個人資訊。如發現已收集，將立即刪除。

7. 第三方服務
   本 App 可能使用第三方服務（如分析工具、廣告、登入服務），其資料收集與使用依各第三方隱私權政策執行。

8. 隱私政策更新
   本政策可能會不定期更新。更新後會於本頁公告，並更新「生效日期」。建議定期查看以了解最新政策。
